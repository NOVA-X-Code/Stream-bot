# 🤖 Telegram Bot - StreamMovies

A professional Telegram bot for streaming video content with advanced features.

## ✨ Features

- 📹 **Video Upload**: Admin users can upload and manage videos
- 🔍 **Smart Search**: Advanced search with filters (genre, year, tags)
- 👥 **Role Management**: User, Admin, and Owner roles
- 🔒 **Security**: Rate limiting, permissions, audit logging
- 📊 **Analytics**: View statistics and activity tracking
- 🚀 **Scalable**: Dockerized with Redis caching

## 🛠️ Tech Stack

- Python 3.11+
- python-telegram-bot 20.3
- Supabase (PostgreSQL)
- Redis for caching
- Docker for deployment

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/NOVA-X-Code/Stream-bot.git
cd Stream-bot
```

### 2. Create virtual environment

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or
venv\Scripts\activate  # Windows
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure environment
```bash
cp bot.env.example bot.env
# Edit bot.env with your credentials
```

### 5. Run the bot
```bash
python -m src.bot
```
